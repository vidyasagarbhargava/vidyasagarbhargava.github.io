## Types of Variables :-    

1. Quantitative Variable (Continuos or discrete)
2. Qualitative Variable (Always discrete)
  * Nominal -> No Order (e.g. State or region)
  * Ordinal -> Order  (e.g. Movie Rating)
 
 
#### Ways of Quantitatively describing the data :-  
1. Graphics Presentation
2. Summary 
3. Tabular

#### Graphics Presentation**

**Single Categorical Variable :-**
1. Bar Graphs
2. Pie Chart

**For Single Quantitative Variable:-**  

1. Box plot  
2. Histogram  

**For Multiple Variables:-**  
1. Scatter plot (Two Quantitative Variables)  
2. Box plots (One Categorical and One Quantitative Variable)  
3. Contigency tables(Two Categorical Variables)
